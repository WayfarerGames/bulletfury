# Wayfarer Games Bulletfury

This folder is the Unity Package Manager source for `com.wayfarergames.bulletfury`.

## Install from Git

Add this to `Packages/manifest.json` in a Unity project:

```json
"com.wayfarergames.bulletfury": "https://github.com/<your-org>/<your-repo>.git?path=/PackageSource/com.wayfarergames.bulletfury"
```
